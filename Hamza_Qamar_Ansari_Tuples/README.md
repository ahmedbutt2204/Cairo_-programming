# Assignment: Tuples in Cairo

## Problem Statement
Write a Cairo program that demonstrates the use of tuples. Create a tuple with three elements of different types, access each element, and print them.

## Requirements
- Declare a tuple with three elements (e.g., felt, bool, felt).
- Access and print each element.

## Sample Output
```
Tuple: (7, true, 15)
First: 7
Second: true
Third: 15
```

## Instructions
- Edit `tuples.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.

## How to Run
```sh
scarb run
```
