# Assignment: Traits in Cairo

## Problem Statement
Write a Cairo program that defines a trait with a function, implements it for a struct, and demonstrates its use.

## Requirements
- Define a trait `Describable` with a function `describe`.
- Implement the trait for a struct (e.g., Book).
- Create an instance and call `describe`.

## Sample Output
```

```

## Instructions
- Edit `traits.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.
