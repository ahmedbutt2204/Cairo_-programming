# Assignment: Structs in Cairo

## Problem Statement
Write a Cairo program that defines a struct to represent a student (name, roll number, marks). Create an instance of the struct, assign values, and print them.

## Requirements
- Define a struct `Student` with fields: name (felt), roll_no (felt), marks (felt).
- Create an instance and assign values.
- Print the struct fields.

## Sample Output
```
Student: { name: 1, roll_no: 22023, marks: 95 }
```

## Instructions
- Edit `structs.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.

## How to Run
```sh
scarb run
```
