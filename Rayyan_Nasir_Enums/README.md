# Assignment: Enums in Cairo

## Problem Statement
Write a Cairo program that defines an enum to represent traffic light states (Red, Yellow, Green). Create a variable of this enum type, assign a value, and print it.

## Requirements
- Define an enum `TrafficLight` with variants: Red, Yellow, Green.
- Create a variable and assign a value.
- Print the enum value.

## Sample Output
```
Current light: Green
```

## Instructions
- Edit `enums.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.

## How to Run
```sh
scarb run
```
