# Assignment: Data Types in Cairo

## Problem Statement
Write a Cairo program that demonstrates the use of different data types (felt, bool, arrays, tuples). Show how to declare, initialize, and print them.

## Requirements
- Declare variables of at least three different data types.
- Initialize them with values.
- Print their values.

## Sample Output
```
num = 42
flag = true
tuple = (5, false)
```

## Instructions
- Edit `data_types.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.

## How to Run
```sh
scarb run
```
