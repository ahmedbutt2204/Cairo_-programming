# Assignment: Functions in Cairo

## Problem Statement
Write a Cairo program that defines and uses functions. Create a function that takes two numbers as input and returns their sum. Call this function from `main` and print the result.

## Requirements
- Define a function `add` that takes two felts and returns their sum.
- Call `add` from `main` with sample values.
- Print the result.

## Sample Output
```
add(5, 7) = 12
```

## Instructions
- Edit `functions.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.

## How to Run
```sh
scarb run
```
